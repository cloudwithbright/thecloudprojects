# Oracle Cloud Infrastructure (OCI) Projects

Welcome to Oracle Cloud projects! Build high-performance applications on Oracle's enterprise-grade cloud infrastructure.

## About Oracle Cloud Infrastructure

Oracle Cloud Infrastructure (OCI) provides high-performance computing, storage, and networking capabilities with enterprise-grade security and governance. OCI is designed for mission-critical workloads and offers competitive pricing with generous free tier offerings.

## Why Learn Oracle Cloud?

- **Enterprise Performance**: Optimized for Oracle databases and enterprise applications
- **Cost Effective**: Competitive pricing with sustained discounts
- **Generous Free Tier**: Always-free resources including compute and databases
- **Security First**: Advanced security features and compliance certifications
- **High Performance**: Bare metal and VM instances with high I/O performance

## Coming Soon

- **Networking**: Virtual Cloud Networks (VCN) and load balancers
- **Storage**: Object storage, block volumes, and file storage

## OCI Free Tier

Oracle Cloud offers an always-free tier including:

- **Compute**: 2 AMD-based compute VMs or 4 Arm-based Ampere A1 cores
- **Storage**: 200 GB total block volume storage
- **Object Storage**: 20 GB
- **Databases**: 2 autonomous databases (20 GB each)
- **Load Balancer**: 1 flexible load balancer
- **Monitoring & Notifications**: Included
