# Google Kubernetes Engine (GKE) Projects

Welcome to GKE projects! Run managed Kubernetes on Google Cloud with industry-leading automation, security, and integration.

## About GKE

Google Kubernetes Engine (GKE) is Google Cloud's managed Kubernetes service. As the creators of Kubernetes, Google offers a production-ready platform with Autopilot and Standard modes, built-in security, and deep integration with GCP services.

## Why Use GKE?

- **Kubernetes Origin**: Built by the same team that created Kubernetes
- **GKE Autopilot**: Fully managed with no node management; pay only for pod resources
- **Security**: Workload Identity, Binary Authorization, and automatic security updates
- **Multi-cluster**: Fleet management and multi-cluster ingress
- **Integration**: Native integration with Cloud Run, BigQuery, and Vertex AI

## Getting Started

1. **Create a GCP account**: [Google Cloud Free Tier](https://cloud.google.com/free)
2. **Enable GKE**: [Kubernetes Engine API](https://console.cloud.google.com/apis/library/container.googleapis.com)
3. **Install gcloud and kubectl**: [Google Cloud SDK](https://cloud.google.com/sdk/docs/install)
4. **Explore docs**: [GKE Documentation](https://cloud.google.com/kubernetes-engine/docs)
