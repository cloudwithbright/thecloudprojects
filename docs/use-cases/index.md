# Use Case Projects

Real-world use case projects that combine multiple cloud services to solve common business and technical scenarios. Each use case includes architecture, implementation steps, and code.

## What You'll Find

Use case projects show how to build complete solutions such as:

- **Web applications** — Frontend, API, database, and CI/CD
- **Event-driven systems** — Message queues, serverless functions, and workflows
- **Data pipelines** — Ingestion, processing, and storage across services
- **DevOps workflows** — Infrastructure, deployment, and monitoring

## By Cloud Provider

| Provider | Description |
|--------|-------------|
| [AWS](/use-cases/aws/) | Use cases built with Lambda, ECS, API Gateway, and other AWS services |
| [Azure](/use-cases/azure/) | Use cases using Functions, Container Apps, Logic Apps, and Azure services |
| [GCP](/use-cases/gcp/) | Use cases with Cloud Run, Cloud Functions, and GCP services |

## Project Structure

Each use case project includes:

- **Scenario** — Business or technical problem
- **Architecture** — Diagram and component overview
- **Prerequisites** — Accounts, tools, and permissions
- **Implementation** — Step-by-step guide
- **Code** — Samples and configuration
- **Cleanup** — How to tear down and avoid ongoing cost
